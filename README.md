## 	:toolbox: Getting Started

<!-- Prerequisites -->

<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/jasontranpk/project-nodejs-shop
```

Go to the project directory

```bash
  cd project-nodejs-shop
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
